# 化療 21 天身體波動照護日曆

一個純前端、零依賴的靜態網頁小工具，協助化療病患與照顧者掌握療程後 21 天內身體狀況的變化（嘔吐高峰、白血球最低點、癌因性疲憊等），並提供對應的照護建議與警示。

## 功能

- 選擇化療開始日期，自動排出完整 21 天月曆
- 依 5 個階段標色顯示：嘔吐最高峰（第1-2天）、延遲緩衝期（第3-5天）、過渡日（第6天）、白血球最低點（第7-14天）、黃金恢復期（第15-21天）
- 疊加癌因性疲憊（CRF）三波高峰資訊（急性疲憊期、骨髓抑制期、回溫期）
- 點選/滑鼠移到某一天，顯示當天詳細的身體狀況說明、照護重點，以及白血球低點期間的發燒急症警示
- 標示「今天」方便對照目前所處階段
- 一鍵匯出 `.ics` 行事曆檔案，可直接匯入 Google 日曆

## 使用方式

不需安裝任何東西，直接用瀏覽器開啟 `index.html` 即可使用；也可以部署到 GitHub Pages 或任何靜態網頁託管服務。

1. 選擇化療開始（第一天施打）的日期
2. 按「開始排程」
3. 將滑鼠移到（手機版為點擊）任一天，查看該天的身體狀況與照護建議

## 匯出至 Google 日曆

在 `index.html` 排好日曆後：

1. 點擊「匯出至 Google 日曆 (.ics)」下載檔案
2. 前往 Google 日曆網頁版（calendar.google.com）
3. 右上角齒輪「設定」→ 左側選單「匯入與匯出」→「匯入」
4. 選擇剛下載的 `.ics` 檔案，並指定要加入的日曆

即可一次匯入全部 21 天的照護行程（建議匯入到另外新建的日曆，方便之後整批刪除或重新匯入）。
